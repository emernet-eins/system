EMERNET E.I.N.S

EMERNET E.I.N.S (Emergency Information Network System) wurde erschaffen um wichtige Informationen im Falle eines absoluten Netzwerkausfalls zur Verfügung zu stellen.
EMERNET zeigt Informationen wie z.B. wichtige Telefonnummern, Notfallnachrichten-Apps (z.B. NINA) und Notfalldienste (z.B. Schutzbunker).
Alle angezeigten Informationen werden durch die Open Source Community auf GitHub und EMERNET-EINS.org verwaltet.
EMERNET wird regelmäßig aktualisiert um die aktuellsten Informationen zu bieten.
Um eine eigene Instanz von EMERNET E.I.N.S anzubieten, besuchen Sie bitte die EMERNET-EINS.org Website, oder das GitHub Profil.
