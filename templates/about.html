EMERNET E.I.N.S - Emergency Information Network System

EMERNET E.I.N.S

EMERNET E.I.N.S (Emergency Information Network System) was created to provide important information in case of a complete communication network failure. EMERNET provides information like important Telephone numbers, emergency news applications (ex. NINA) and emergency services (ex. shelter locations). All information provided is managed by the Open Source Community on GitHub and EMERNET-EINS.org. EMERNET is frequently updated to provide the latest information. To host your own instance of EMERNET E.I.N.S, please refer to the official EMERNET website, or visit the GitHub organisation.
